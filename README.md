# How AI Tools Improve Productivity in Development Workflows

Interactive presentation exploring how AI moves beyond code completion to transform the entire development lifecycle — from autonomous agents to platform-level orchestration.

## Overview

Built as a single-file HTML presentation with scroll-snap navigation, animated data visualizations, and expandable content cards. Designed for live delivery via screen share.

### Sections

1. **The Challenge** — Why the productivity ceiling isn't code, it's everything around it
2. **The Framework** — Three layers of AI in the development lifecycle
3. **From Experience** — Real-world results from federal and startup environments
4. **The Honest Conversation** — Trust, governance, and ROI at enterprise scale

## Tech

- Vanilla HTML/CSS/JS — no build step, no dependencies
- [Tailwind CSS](https://tailwindcss.com/) via CDN
- [Lucide Icons](https://lucide.dev/) via CDN
- [Mona Sans](https://github.com/github/mona-sans) + [Hubot Sans](https://github.com/github/hubot-sans) typefaces
- Intersection Observer API for scroll-triggered animations
- CSS scroll-snap for section navigation

## Navigation

- **Scroll** or **↑↓ arrow keys** to navigate between sections
- **Click cards** to expand detailed content
- **Navigation dots** (right side) for direct section access
